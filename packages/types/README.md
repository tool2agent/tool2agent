# @tool2agent/types [![API docs](https://img.shields.io/badge/API%20docs-blue)](https://tool2agent.org/docs/)

TypeScript type definitions for the tool2agent protocol. This package is for tool2agent tooling developers. Agent developers should use tool2agent bindings for existing LLM frameworks, e.g. [AI SDK bindings](https://github.com/tool2agent/tool2agent/tree/master/packages/ai)

## Install

```bash
pnpm add -D @tool2agent/types
```
